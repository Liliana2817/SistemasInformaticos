Ejercicio 3
1. Comprobar el estado del repositorio.
2. Crear un fichero indice.txt con el siguiente contenido:
Capítulo 1: Introducción a Git
Capítulo 2: Flujo de trabajo básico
Capítulo 3: Repositorios remotos
3. Comprobar de nuevo el estado del repositorio.
4. Añadir el fichero a la zona de intercambio temporal.
5. Volver a comprobar una vez más el estado del repositorio.

$ git status
$ git add indice.txt
$ echo -e Capítulo 1: Introducción a Git Capítulo 2: Flujo de trabajo básico Capítulo 3: Repositorios remotos > indice.txt
$ git status
$ git add  indice.txt
$ git status

Ejercicio 4
Realizar un commit de los últimos cambios con el mensaje
“Añadido índice del libro.” y ver el estado del repositorio.
$ git commit -m "Añadido indice del libro."
$ git add indice.txt 
$ git status

Ejercicio 6
